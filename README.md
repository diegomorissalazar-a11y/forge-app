# MELQART v200 — Design System

Versión de rediseño integral con modo claro/oscuro y sistema visual inspirado en los tableros entregados.

## Incluye
- `index.html`
- `styles.css`
- `app.js`
- `assets/brand/`
- `assets/icons/`

## Cambios principales
- Modo claro y modo oscuro con `html[data-theme]`.
- Botón de cambio de tema en topbar.
- Paleta principal: púrpura fenicio, oro antiguo, bronce cálido, turquesa atenuado, carbón, marfil y arena clara.
- Sistema de iconos SVG en `assets/icons/`.
- Navegación inferior con iconos SVG en vez de emojis.
- Componentes base alineados a cards, pills, chips, barras y tablas del tablero.
- Se mantiene lógica base: Firebase, auth, localStorage, rutinas y render existentes.

## Subida a GitHub Pages
Sube el contenido completo del ZIP a la raíz del repo.

## Nota
El rediseño es incremental sobre la base funcional para cuidar rendimiento. La iconografía interna más profunda puede seguir migrándose pantalla por pantalla en versiones siguientes.
